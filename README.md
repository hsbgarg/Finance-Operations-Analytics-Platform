# 💰 Finance Operations Analytics Platform

An end-to-end Finance Operations Analytics Platform built using **Python, Streamlit, Pandas and Plotly**.

This project simulates a real-world finance analytics system used by NGOs, healthcare organizations and consulting firms to monitor budgets, grants, vendors, employees and financial transactions.

---
